JXKit Reference JXEnv Protocol Reference

JXEnv

public protocol JXEnv : AnyObject

A JXEnv is an abstraction of a JavaScript execution environment. The associated ValueType represents the value encapsulationd type.

  • installModule(named:in:) Extension method

    Runs a script in the standard bundle module location "Resources/JavaScript".

    Throws

    an error if the script could not be located or if an error occured when running

    Example:

    /// Installs the `esprima` module used by `JavaScriptParser`.
/// This will execute the bundle's `Resources/esprima.js` resource.
public func installJavaScriptParser() throws {
    try installModule(named: "esprima", in: .module)
}

    Declaration

    Swift

    public func installModule(named name: String, in bundle: Bundle) throws -> JXValType

    Parameters

    name

    the name of the script to run (not including the “.js” extension)
    bundle

    the bundle from which the load the script (typically Bundle.module for a Swift package)
