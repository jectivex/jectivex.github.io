JXContextGroup

open class JXContextGroup
extension JXContextGroup : JXVM

A JavaScript virtual machine that is used by a JXContextGroup instance.

JXValue references may be used interchangably with separate JXContext instances that created from the same JXContextGroup, but sharing between different JXContextGroups will result in undefined behavior.

Note

This wraps a JSContextGroupRef, and is the equivalent of JavaScriptCore.JSVirtualMachine
